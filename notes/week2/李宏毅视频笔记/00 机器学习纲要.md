# 机器学习基本概念

机器学习 $\approx$ 机器自动找一个函式

## 根据函式输出来分类

- 回归（Regression）：函式的输出是一个数值
- 分类（Classification）：函式的输出是一个类别
- 结构化学习（Structured Learning）：生成有结构的物件
  - 又叫做生成式学习（Generative Learning）

# 找出函式的三步骤

设定范围 $\rightarrow$ 设定目标 $\rightarrow$ 达成目标

前置作业：决定要找什么样的函数

## 设定范围
定出候选函式的集合（Model）
- 深度学习中类神经网络的结构（例如：CNN，RNN，Transformer等）指的就是不同的候选函式的集合
- 数据越少，通常越需要限制模型；数据越多，才越有能力使用复杂模型​

## 设定标准
定出`评量函式好坏`的标准（Loss）
- Loss Function 取决于训练资料
  - Supervised Learning
  - Semi-supervised Learning
- 在 Training 上训练出来的函式在 Testing 上不一定好
- 在 Loss 上做额外考量（如：Regularization）

## 达成目标
找出最好的函式 $\rightarrow$ 最佳化（Optimization）
$$
f^* = arg \min_{f \in \mathcal{H}} L(f)
$$
- 通过最佳化演算法（Optimization Algorithm）
- 需要先设定一系列超参数（Hyperparameter）
  - 包括 Learning Rate，Batch Size，How to Init
- 一个好的演算法应该对超参数不敏感

## 相互作用
有一些好的方法是可以支援其他步骤的
- 设定范围 Residual Connection，Batch Normalization
- 设定标准 Cross-entropy loss
