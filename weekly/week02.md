## 一、本周任务

1. 学习机器学习基本原理，完成李宏毅机器学习课程 P13—P21 相关内容的笔记整理。
2. 学习生成式学习、工具型 AI、深度学习、梯度下降和 CNN 等基础知识。
3. 学习 Linux 环境配置相关资料与 Linux 基本命令。

## 二、完成情况

- 机器学习基本原理学习
  - 理解机器学习可以概括为“机器自动找一个函数”；
  - 梳理了回归、分类、结构化学习三类任务的区别；
  - 总结了机器学习找函数的三步骤：设定范围、设定标准、达成目标；
  - 了解了 Model、Loss、Optimization、Hyperparameter 等核心概念；
  - 认识到训练集上的表现和测试集上的表现不一定一致，需要关注泛化能力。
- 生成式学习与工具型 AI
  - 学习了 Autoregressive Model 和 Non-autoregressive Model 两种生成策略；
  - 比较了 AR Model 与 NAR Model 在生成速度、品质和应用场景上的差异；
  - 了解了 New Bing、WebGPT、Toolformer 等使用搜索引擎或外部工具的 AI 形态；
  - 认识到工具调用也可以被建模为特殊符号序列上的“文字接龙”。
- 深度学习基础
  - 梳理了深度学习从感知机、多层感知机、反向传播到 GPU 时代的发展脉络；
  - 学习了神经网络的基本组成：输入层、隐藏层、输出层；
  - 理解了网络参数由神经元中的权重和偏置组成；
  - 了解了全连接前馈网络、总损失函数、梯度下降和反向传播的基本作用。
- 梯度下降与优化
  - 学习了学习率对训练过程的影响；
  - 了解了自适应学习率、随机梯度下降和特征缩放的基本思想；
  - 认识到梯度下降可能遇到局部最小值、鞍点和平坦区域收敛缓慢等问题。
- CNN 与图像分类
  - 理解图像可以表示为长、宽、通道组成的三维张量；
  - 学习了感受野、参数共享、卷积层、特征图和池化等概念；
  - 了解了典型 CNN 流程：Convolution → Pooling → Flatten → Fully Connected → Softmax。
- Linux 基本命令学习
  - 学习了 shell、终端仿真器、文件系统树和当前工作目录等概念；
  - 掌握了 `pwd`、`ls`、`cd`、`file`、`less` 等基础命令；
  - 学习了通配符、`mkdir`、`cp`、`mv`、`rm`、`ln` 等文件和目录操作；
  - 了解了 `type`、`which`、`help`、`man`、`apropos`、`whatis`、`info` 等命令查询方式；
  - 初步学习了标准输入、标准输出、标准错误和输出重定向。

## 三、本周学习收获

1. 对机器学习任务有了更系统的认识：先确定函数集合，再定义衡量标准，最后通过优化方法找到较好的函数。
2. 理解了生成式学习并不是单一方法，而是可以根据任务特点在逐步生成和一次生成之间组合选择。
3. 初步建立了深度学习的整体框架，知道神经网络结构、损失函数和优化算法分别承担什么角色。
4. 认识到梯度下降并不只是套公式，学习率、特征缩放、随机采样和优化地形都会影响训练效果。
5. 了解 CNN 通过感受野和参数共享减少图像任务中的参数量，并用卷积与池化提取局部特征。
6. 熟悉了 Linux 命令行的基本工作方式，为后续在 WSL2、Jupyter Notebook 和 PyTorch 环境中学习打基础。
